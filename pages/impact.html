How it works

  1. Users opt in inside the app — nothing leaves the device without consent.
  2. On opt-in, the user downloads a local JSON file (k-anonymised snapshot).
  3. To share, they submit it as a GitHub pull request using a pre-filled link.
  4. The maintainer runs the freeze script which merges contributions and regenerates this page.
