## Usage notes

- **Personalise the opening line** for each recipient — reference a specific post, publication, or event they were involved in.
- **Attach or link the EACEA one-pager** if sending as InMail (see `docs/outreach/eacea-one-pager.html` — print to PDF from a browser).
- **Connection request notes** have a 300-character limit on LinkedIn — always use the short version for initial connection requests.
- **Follow up** only once, after 7-10 days, with a brief note referencing the original message.
- **Do not mass-send** — each message should feel individually written. Target 3-5 contacts maximum in the Youth Unit.
