Turkey is a Programme Country for KA220-YOU, which satisfies eligibility. Timeline: concept-note shared in May 2026, draft proposal ready by January 2027, submission for the **5 March 2027 deadline**, 24-month project starting Q3 2027. **Q2: "What happens to the tool and its IP after the grant?"**

All code and data stay **MIT open-source**, hosted on GitHub, free-to-use for any participant or partner. The KA220 grant funds staff time, youth testing, dissemination, and the Open API / Impact Dashboard buildout — it does not buy exclusivity. Post-grant, a separate commercial tier (premium features only) may be explored under Superior Boy's startup arm, but the core companion and the grant-funded deliverables remain MIT and publicly available in perpetuity. This framing is explicit in the concept note so reviewers see no conflict between the non-profit grant mission and the founder's long-term sustainability model.
